## How to Run
- **Set Up**: Install required R packages:
  `kknn`, `caret`, `glmnet`, `kernlab`, `geosphere`, `neuralnet`.
- **Execution**: Run `.r` scripts in respective lab folders.
- **Reproducibility**: Use `set.seed(12345)` for consistent results.
